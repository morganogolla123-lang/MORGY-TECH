### 🐍 **Pokémon Commands**

| Command | Description |
|---------|-------------|
| `.pokemon <name>` | Get Pokémon stats & artwork |
| `.randompokemon` | Random Pokémon with artwork |
| `.pokemontypes` | List all 18 types |
| `.pokemonbytype <type>` | Pokémon by type |

---

### 🎬 **TV Shows & Movies**

| Command | Description |
|---------|-------------|
| `.tvsearch <name>` | Search TV shows with poster |
| `.tvshowinfo <id>` | Full show details & poster |
| `.tvepisodes <id>` | Episode list |
| `.tvschedule <country>` | Today's TV schedule |
